# 检验报告解读参考（示例）

以下为示例摘录，实际使用时请替换为院内认可的规范或指南原文摘录。

## 危急值

- 血钾 <2.5 或 >6.5 mmol/L
- 血糖 <2.8 或 >22.2 mmol/L
- 血红蛋白 <50 g/L（急性贫血）
- 血小板 <30×10⁹/L（出血风险）
- 白细胞 <1.0×10⁹/L（严重感染风险）

出现上述情况需在解读中首先强调并建议立即临床处置。

## 肝功能异常简要参考

- ALT/AST 轻度升高：常见于脂肪肝、药物、慢性肝炎等，需结合胆红素、白蛋白等综合判断。
- 胆红素升高：区分溶血性、肝细胞性、梗阻性，结合 ALP、GGT 等。

## 肾功能

- 肌酐升高需结合 eGFR 评估慢性肾病分期；急性升高需排除 AKI 及可逆因素。
